# 战前准备

从[系统商店](xi-tong-shang-dian.md)或[玩家市场](wan-jia-shi-chang.md)内购买，或者[从私人仓库中取出](si-ren-cang-ku.md)各种所需要的战斗用品

通常来说，你应该携带

* 适量的武器（远程武器、近战武器、首领信物）
* 适量的防具（头盔、胸甲、护腿、靴子）
* 适量的且有需要用到的消耗品（食物、医疗用品、弹药、投掷物、药水）

如果你想要赚取更多的钱财，也可以携带钥匙去开启[钥匙房](../../tie-men-zen-me-da-kai.md)。而且钥匙是唯一一种不会在你死亡时掉落的物品，不管是局外带入的还是局内获得的都会受到保护，但你可以选择主动丢掉它

<mark style="color:red;">请不要将你的仓库兑换券或者变卖物带入局内，这会在你死亡时掉落</mark>

检查好你的携带物品后，你就可以[进行下一步](../zu-dui-xi-tong.md)了
